Description:

Abstract:

This dataset contains the taxmap parcels for Suffolk County covering approximately 920 square miles. Real Property Tax Service Agency (RPTSA) is the sole authority responsible for maintaining the official Suffolk County Tax Map for ad valorem purposes. Each parcel on a tax map is uniquely identified by a District, Section, Block and Lot number. The tax map number is the keystone to the description of real property in Suffolk County. This dataset excludes condominiums and overlap parcels. To obtain the full taxmap landbase composition in Suffolk County, this dataset should be used in conjunction with the Overlap and MultiUnit feature classes

Purpose:

RPTSA maintains a tax map base and a database of attribute data, commonly referred to as AREIS (Advanced Real Estate Information System). This dataset was created for the express purpose of sharing the Suffolk County AREIS data in a Geographic Information System (GIS) format with other government entities as part of the County's goal to create better integrated layers of GIS

Data_Quality_Information:

Attribute_Accuracy:

Attribute_Accuracy_Report: 95% accurate to Microstation files

Logical_Consistency_Report:

Basemap maintained by entering parcels into a digital format directly from surveys and deeds using coordinate geometry which is then tied to the geodetic reference framework. RPTSA drafters use survey data and deed descriptions specifying the bearing and distance between end points of the parcel boundary lines. The individual parcels are then joined together in a work file to form a small "local" parcel network that is then defined in terms of all the current dimensional information and coordinates which identify the relative position of parcel end points within the network. The work file (the parcel network) is rotated into the production file (the "regional" map section file) containing the control coordinate system (Stateplane NAD 27) and using known survey control points (USGS /Suffolk County control monuments) and road ROW lines. This process of rotation is repeated as new and additional known survey points or new information from deed transfers are introduced into the parcel network. Therefore the parcel features have greater relative accuracy than absolute accuracy.

Completeness_Report:

This dataset is complete to 2008. As RPTSA drafters edit the Microstation files for the purposes of incoporating new subdivisions, splits, assemblages, or lot line adjustments, the map sections are converted to this format. No greater than 1 month lag time between when the official tax map was edited and the map section(s) converted to SDE

Positional_Accuracy:

Horizontal_Positional_Accuracy:

Horizontal_Positional_Accuracy_Report:

For individual parcel source information refer to the official copy of the tax map and deeds. The tax map files contain information on parcel boundaries, whether it is precision linework (mapped according to deed and rotated into geodetic framework), digitized from the mylars by Bowne as described in Process Step 4, or NY State Plane precision. Accuracy can be as high as 1/8 of an inch. Part 189 of Suffolk County regulations regarding preparation of the tax maps by authority of the Real Property Tax Law states that the New York State Plane coordinate indications on each tax map section must be accurate to within 10 ft of the true grid value. The dimensions of the parcel are not intended to be as accurate as, or a replacement of, a survey.
